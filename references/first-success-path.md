# First Success Path

## 你要准备什么
每个飞书机器人至少提供：
- `accountId`
- `appId`
- `appSecret`
- `botName`

## 你怎么开始
直接说：

```text
我已经创建好了这批飞书机器人，帮我接成 OpenClaw 多 Agent，先给我预览，再执行。
```

## 你会得到什么
- 一份多 Agent 规划预览
- 建议新增的 `agents / accounts / bindings`
- 建议创建的 `workspace / agentDir`
- 确认后的安全落地结果
- 巡检结果与下一步测试建议

## 怎么确认第一次成功
至少满足：
- 配置成功写入
- 目录成功创建
- 巡检无致命错误
- 至少 1 个新 bot 完成一次真实回复验证
